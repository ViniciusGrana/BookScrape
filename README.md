# BookScrape: Automação de Coleta de Dados Web

O **BookScrape** é uma ferramenta de automação desenvolvida em Python para extração sistemática de dados de catálogos de livros online (Web Scraping), permitindo a coleta e exportação de informações para análise posterior.

## 🚀 Tecnologias e Bibliotecas

O projeto utiliza o ecossistema Python para automação e manipulação de arquivos:

| Biblioteca | Finalidade |
| :--- | :--- |
| **BeautifulSoup4** | Parsing de HTML e navegação na árvore DOM |
| **Requests** | Requisições HTTP para obtenção do conteúdo das páginas |
| **Openpyxl** | Exportação dos dados extraídos para o formato Excel (`.xlsx`) |
| **Python 3** | Lógica de automação e tratamento de dados |

## 🛠️ Funcionalidades Principais

A ferramenta automatiza o processo manual de coleta de informações:

*   **Extração Automatizada:** Coleta de títulos, preços, disponibilidade e avaliações de livros.
*   **Tratamento de Dados:** Limpeza de strings e formatação de valores numéricos durante o scrape.
*   **Persistência de Dados:** Geração automática de arquivos de planilha para análise estruturada.

## 🎓 Contexto do Projeto

Este projeto demonstra habilidades em **Automação de Processos**, **Manipulação de DOM** e **Integração com Arquivos Externos**, competências fundamentais para um desenvolvedor de sistemas moderno que precisa lidar com integração de dados de fontes externas.
